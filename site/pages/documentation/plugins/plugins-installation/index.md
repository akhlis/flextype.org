## Steps to install plugins{.margin-top-hard}

1. Unzip plugin to the folder `/plugins/`
2. Go to `/config/system.yml` and add plugin name to plugins section.
3. Save your changes.

Example:
```
...
# Site Plugins
plugins:
  myplugin
```
