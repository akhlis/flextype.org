--- title: Requirements description: Documentation for Flextype Developers and Flextype Users. template: documentation ---

Make sure your server meets the following requirements.

  • Webserver (Apache with Mod Rewrite)
  • PHP 7.1.3 or higher with PHP's Multibyte String module

For better performance we recommend to use caching: APC, APCu, XCache, Memcached, or Redis.
