--- title: Hooks description: Documentation for Flextype template: documentation ---

With hooks you can intercept all kinds of Flextype events to inject your own code.

Here is a list of available hooks:

onCurrentEntryBeforeProcessed
onCurrentEntryBeforeDisplayed
onCurrentEntryAfterProcessed
onPluginsInitialized
onShortcodesInitialized
onThemeMeta
onThemeHeader
onThemeFooter
