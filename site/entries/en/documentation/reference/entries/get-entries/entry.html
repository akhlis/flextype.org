Documentation / Reference / Entries / Entries::getEntries()

Get entries

getEntries(string $url = '', string $order_by = 'date', string $order_type = 'DESC', int $offset = null, int $length = null, bool $multilevel = false, bool $raw = false) : array

Parameters

Name Type Default Description
url string Page url.
order_by string date Specific entry field.
order_type string DESC Order type: DESC or ASC
offset int null Offset
length int null Length
multilevel bool false Get nested entries or not.
raw bool false Parse content or raw content without parsing.

Return type

array

Examples

Get list of entries for entry projects
$entries = Entries::getEntries('projects');
Get list of entries for entry projects and order them by date with order type DESC
$entries = Entries::getEntries('projects', 'date', 'DESC');
