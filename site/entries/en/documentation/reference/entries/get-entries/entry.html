Get entries

getEntries(string $url = '', string $order_by = 'date', string $order_type = 'DESC', int $offset = null, int $length = null, bool $multilevel = false, bool $raw = false) : array

Parameters

Name Type Default Description
url string Page url
order_by string date Order by specific entry field
order_type string DESC Order type (DESC or ASC)
offset int null Offset
length int null Length
multilevel bool false Get nested entries
raw bool false Raw content

Return type

array

Examples

Get list of posts for entry blog
$posts = Entries::getEntries('blog');
Get list of posts for entry blog and order them by date with order type DESC
$posts = Entries::getEntries('blog', 'date', 'DESC');
Display list of posts in the template
<?php foreach($posts as $post): ?>
    <a href="<?= $post['slug'] ?>"><?= $post['title'] ?></a>
<?php endforeach ?>
