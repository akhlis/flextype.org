Get entry

getEntry(string $url = '', bool $raw = false, bool $hidden = false)

Parameters

Name Type Default Description
url string Page url
raw bool false Raw content
hidden bool false Hidden entry

Return type

array | string

Examples

Get specific post from blog
$post = Entries::getEntry('blog/hello-world');
Display post field in the template
<?= $post['title'] ?>
Get specific post and display post field in the template
<?= Entries::getEntry('blog/hello-world')['title'] ?>
